-에이전트 이름 : 언어 학습 에이전트

-한 줄 소개: 내가 가진 단어 목록(csv파일)을 ai가 보고 매일 한 단어씩 학습시켜주는 에이전트

-핵심 기능 3가지: 
1. 단어 선택 에이전트 - 히스토리에 저장된 사용자의 지난 기록을 파악해서 학습할 단어 선택
2. 상황 제시 에이전트 - 단어가 쓰일 수 있는 상황을 제시해주고 그 상황에 어울리는 한 문장을 단어를 포함하여 입력한다.
3. 피드백 에이전트 - 사용자가 단어를 잘 이해하고 사용했는지, 문맥에 맞게 잘 사용했는지 등을 피드백해주고, 정답 문장도 제시해준다.
